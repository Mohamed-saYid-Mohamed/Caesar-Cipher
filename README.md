⚠ *Warning: Educational Purpose Only / Digniin: Ujeeddo Waxbarasho Kaliya*

This project demonstrates a simple Caesar Cipher encryption and decryption.
It is intended for learning purposes and *not secure for real-world encryption*.
Use at your own risk.

Mashruucan wuxuu muujinayaa sida loo isticmaalo *Caesar Cipher* si qoraalka loo qarxiyo (encryption) loona furo (decryption).
Waxaa loogu talagalay *ujeeddo waxbarasho* kaliya, mana aha mid ammaan u ah xogta dhabta ah ee internet-ka.
